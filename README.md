__Tribute Page - Nikola Tesla__
Welcome to my first official web development project! This is a tribute page dedicated to Nikola Tesla, the legendary inventor and electrical engineer who revolutionized our world.

__About This Project__
This project was created as part of my learning journey in web development. The main goal was to practice and apply my knowledge of HTML5 and CSS3 to create a responsive and visually appealing web page.

__Features__
Biographical Information: A brief overview of Nikola Tesla’s life and major contributions.   

Visual Highlights: Includes images of Tesla and some of his most significant inventions.

Key Facts Table: A structured section highlighting why Tesla remains a visionary in the scientific community.

Interactive Elements: Features a comment form and links to learn more about his work.

Responsive Design: Styled to look clean and structured across different screen sizes.

__Technologies Used__
HTML5: For structuring the content.

CSS3: For styling, layouts, and adding visual flair (including gradients and hover effects).

__My Learning Journey__
Since this is my first official project in web development, I focused on:

Understanding the structure of an HTML document.

Linking external CSS files.

Using CSS properties like Flexbox, positioning, and background styling to improve the user interface.

Learning how to host and manage code on GitHub.
